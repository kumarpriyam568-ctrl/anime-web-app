# Anime Recommender Web App 🎌

A simple Flask web app that recommends anime based on mood.

## Features
- User input for mood
- Displays anime recommendations
- Clean UI

## How to Run

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
